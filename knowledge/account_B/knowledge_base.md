# 副業情報局 ナレッジベース

## アカウント概要
- スタイル: クリーン・フレンドリー
- ターゲット: 副業初心者〜始めたばかりの人
- 主力ファネル: アフィリエイトツール紹介 (30%コミッション)

## コンテンツ戦略

### 動画の方向性
1. **ツール紹介系** — 「これ使ってみたら便利だった」系
2. **初心者ガイド系** — わかりやすい解説
3. **体験談系** — リアルな感想・数字

### フックの黄金パターン
- 「ねえ、知ってた？」系 → 親しみやすさ
- 「こっそり教えます」系 → 特別感
- 「正直に言います」系 → 誠実さ

## ファネル設計

### 導線
動画 → プロフィールリンク → ツール紹介LP → 無料登録/購入 → 30%コミッション

### KPIs
- LP CTR目標: 2.5%以上
- LP CVR目標: 5%以上 (無料登録が多いため高め)
- 目標RPV: ¥0.40以上

## 注意事項
- 比較系コンテンツは公平性を保つ
- ステマ・PR表記は必ず行う
- 押しつけがましいCTAは避ける
