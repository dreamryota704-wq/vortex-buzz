# バイラルパターン集 (Viral Patterns)

## 再生数が伸びる動画の共通パターン

### 1. 冒頭3秒の法則
- 最初の3秒で「見続ける理由」を提示する
- 数字・衝撃情報・共感ワードを使う
- 「待って」「これ知らないと損」「実は」などの引き止めワード効果的
- 視聴者の悩みを代弁する言葉が最も強い

### 2. 情報密度とテンポ
- 1動画に1メッセージを絞る（欲張らない）
- テロップのテンポは話すテンポより少し早め
- BGMは80〜120BPMが視聴継続率に良い影響
- 無音区間は0.5秒以内に収める

### 3. 保存される動画の条件
- 「後で使える情報」を含む
- リスト形式（3つのコツ、5つの方法）
- スクショしたくなる「まとめスライド」の存在
- 数字・実績の具体性（「月10万」より「月10万2千円」）

### 4. コメントを誘発するパターン
- 賛否が分かれる意見の提示
- 「あなたはどっち？」の二択
- 「コメントで教えて」の明示的な誘導
- 視聴者の体験談を引き出す質問

### 5. シェア・引用されるパターン
- 「友達に教えたくなる」知識
- 「自分も同じだった」共感要素
- 「嘘だと思ったら本当だった」系
- 業界の常識を覆す情報

## フックの型

### 数字フック
```
月○万円稼ぐ方法、今から教えます
○日で変わった話
○個の方法を全部公開
```

### 共感フック
```
○○で悩んでいる人、見てください
正直に言います、○○って○○です
私も去年まで○○だった
```

### 疑問フック
```
なぜ○○する人は○○なのか
○○がうまくいかない本当の理由
○○を知らずに○○している人へ
```

### 否定フック
```
○○はやめてください
○○してはいけない理由
ほとんどの人が間違えている○○の方法
```

## 高CVR動画の特徴

1. **問題→解決→証拠→CTA** の流れが明確
2. CTAが具体的（「プロフィールリンクから無料で確認」）
3. 視聴者の心理段階に合わせたメッセージ
4. 「今すぐ」の緊急性を含む
5. LP・サービスへの自然な橋渡し

---

## アルゴリズムを味方につけるポイント

- 再生完了率 > 60% を目指す
- 保存率 > 3% が良好指標
- コメント率 > 0.5% が良好指標
- 最初の30分の反応が重要（インプレッション増加のシグナル）
- ハッシュタグは5〜8個が最適（多すぎるとスパム判定）
